Jeu Othello - Java POO

À propos du projet


Le projet Othello est une interprétation du jeu classique développé en Java avec une approche orientée objet. Il comprend une interface utilisateur graphique intuitive et une IA compétitive pour une expérience de jeu engageante.

L'application utilise une architecture MVC pour une structure de code propre et une stratégie d'IA avancée basée sur les algorithmes Min/Max et Alpha/Beta Cutoff pour une expérience de jeu dynamique.

Écran d'accueil du jeu Othello avec une interface utilisateur minimaliste et claire, présentant le plateau de jeu initial.

Capture d'écran montrant le jeu en cours, avec des coups stratégiques effectués par les joueurs et l'IA.

Fin du jeu, où les résultats finaux sont affichés, montrant le gagnant entre les joueurs ou contre l'IA.


Conclusion


Le développement du jeu Othello en Java a été un projet enrichissant qui a démontré la puissance et la flexibilité de la programmation orientée objet. À travers la création d'une interface utilisateur interactive et le développement d'une IA stratégique, ce projet a offert non seulement un défi technique mais aussi une occasion d'approfondir les principes de conception logicielle.


Pour en savoir plus sur les fonctionnalités et le code source du jeu, le projet est consultable sur GitHub.
